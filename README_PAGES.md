# GitHub Pages 预览设置指南

## 方法一：启用 GitHub Pages（推荐）

### 第一步：确认文件已准备

已在 `docs/index.html` 中创建了界面预览文件。

### 第二步：启用 GitHub Pages

1. 访问你的 GitHub 仓库：
   https://github.com/heavenchenggong/stock-cost-manager

2. 点击 **「Settings（设置）」** 标签

3. 在左侧菜单中找到 **「Pages」**

4. 在「Source」部分：
   - **Branch:** 选择 `main` 分支
   - **Folder:** 选择 `/docs` 文件夹
   - 点击「Save」

5. 等待 1-2 分钟，页面会显示你的网站地址：
   ```
   https://heavenchenggong.github.io/stock-cost-manager/
   ```

6. 访问上面的链接，就能看到渲染后的界面了！

---

## 方法二：下载后在本地打开（最简单）

### 步骤：

1. 在 GitHub 仓库中找到 `docs/index.html` 文件

2. 点击「Raw」按钮，查看源代码

3. 右键点击页面 → 「网页另存为」

4. 保存为 `stock-cost-manager.html`

5. 双击打开这个 HTML 文件，就能在浏览器中看到界面了！

---

## 方法三：下载整个项目

1. 在 GitHub 仓库页面
2. 点击绿色按钮「Code」
3. 选择「Download ZIP」
4. 解压后，找到 `docs/index.html`
5. 双击用浏览器打开

---

## 🎯 推荐方法

**最快：** 方法二（下载后在本地打开）- 30 秒搞定

**最方便：** 方法一（GitHub Pages）- 设置一次后，随时能在线查看
